# 🐧 Curso Linux do Zero
## Introdução – Módulo 1: Fundamentos do Linux

---

## 👋 Boas-vindas ao Curso

Seja bem-vindo(a) ao **Curso Linux do Zero**!

Este curso foi criado para **iniciantes absolutos**, especialmente para quem:
- Nunca usou Linux
- Vem do Windows
- Tem curiosidade sobre Linux, mas acha que é “difícil”
- Está começando na área de TI ou formação profissional

👉 **Nenhum conhecimento prévio é necessário.**  
Aqui, você vai aprender **do zero, passo a passo e sem medo**.

---

## 🎯 Objetivo do Curso

O objetivo do curso é:

- Apresentar o Linux de forma **simples e acessível**
- Explicar os conceitos fundamentais com linguagem clara
- Preparar o aluno para usar Linux na prática
- Construir uma base sólida para estudos mais avançados

---

## 📦 Sobre o Módulo 1 – Fundamentos do Linux

Este primeiro módulo é **totalmente conceitual** e tem um papel muito importante:

> ✅ **Criar uma base sólida**  
> ✅ **Eliminar o medo do Linux**  
> ✅ **Organizar o entendimento do aluno**

⚠️ Neste módulo:
- **Não usamos terminal**
- **Não instalamos nada**
- Focamos em **entender antes de fazer**

---

## 🧠 O que você vai aprender neste módulo?

Ao final do **Módulo 1**, você será capaz de:

- Explicar o que é o Linux
- Entender onde o Linux é utilizado
- Compreender o conceito de distribuições
- Entender o que são pacotes `.deb` e `.rpm`
- Reconhecer as principais famílias Linux
- Avançar para a prática com segurança
