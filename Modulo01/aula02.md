## 🟢 Aula 02 – Por onde começar no Linux?

---

## 📌 Descrição Geral

Nesta aula, o aluno aprende **como iniciar no mundo Linux com segurança**, entendendo que não é necessário abandonar o Windows de imediato e que existem várias formas simples de começar a usar Linux.

## 🎯 Objetivos de Aprendizagem

Ao final desta aula, o aluno será capaz de:

- Entender **quem pode usar Linux**

## 🧠 Conteúdo Programático

---

### 1️⃣ Quem pode usar Linux?

Linux **não é exclusivo** para programadores ou especialistas em TI.

Pode usar Linux:

- 👩‍🎓 Estudantes
- 👨‍🏫 Professores
- 🧑‍💼 Usuários domésticos
- 👨‍💻 Profissionais de TI
- 🏢 Empresas

📌 Destaque importante:
> Qualquer pessoa que saiba usar um computador pode aprender Linux.

---

### 2️⃣ Linux é para usuário comum?

Sim ✅

Atualmente o Linux oferece:

- Interface gráfica amigável
- Uso de mouse e teclado
- Navegador de internet
- Editor de textos
- Loja de aplicativos

📌 Comparação direta:
> Se você sabe usar Windows, consegue aprender Linux.

---

### 3️⃣ Formas de Começar no Linux

Existem **três formas principais** de começar a usar Linux.

---

#### 🟢 1. Máquina Virtual (Recomendado para iniciantes)

- Linux roda dentro do Windows
- Não altera o computador
- Ideal para estudo e testes

✅ Vantagens:
- Seguro
- Sem risco de perder arquivos
- Fácil de remover

❌ Desvantagem:
- Um pouco mais lento

---

#### 🟡 2. Dual Boot (Windows + Linux)

- Linux e Windows instalados no mesmo computador
- Escolha do sistema ao ligar o PC

✅ Vantagens:
- Melhor desempenho
- Uso real do Linux

⚠️ Atenção:
> Requer cuidado na instalação (será visto mais adiante no curso)

---

#### 🔵 3. Linux como Sistema Único

- Linux substitui completamente o Windows

✅ Vantagens:
- Melhor desempenho
- Uso total do sistema

⚠️ Recomendado apenas após ganhar experiência.

---
#### 🟣 4 . Live Boot (Teste sem instalar)

- Execute o Linux diretamente a partir de um pendrive ou DVD, sem alterar o disco rígido
- Ideal para experimentar distribuição, testar drivers e ver compatibilidade de hardware

✅ Vantagens:

- Não modifica o sistema atual
- Risco zero para arquivos no disco
- Rápido para testar sem instalar

⚠️ Limitações:

- Desempenho e persistência limitados — configurações e arquivos podem não ser salvos a menos que o pendrive tenha persistência configurada
